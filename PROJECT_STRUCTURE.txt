Quick Start Commands:
====================

Setup:
  npm install                          # Install dependencies
  cp .env.example .env                # Create environment file
  npm run compile                     # Compile contracts

Testing:
  npm test                            # Run test suite
  npm run simulate                    # Run simulation

Deployment:
  npm run deploy                      # Deploy to Sepolia
  npm run verify                      # Verify on Etherscan
  npm run interact                    # Interactive interface

Hardhat Tasks:
  npx hardhat accounts --network sepolia
  npx hardhat contract-info --address CONTRACT_ADDRESS --network sepolia
  npx hardhat register-member --contract CONTRACT --type Premium --network sepolia
  npx hardhat record-workout --contract CONTRACT --calories 500 --duration 45 --intensity 8 --network sepolia
